# Security Policy

## Official Distribution Channels
To protect yourself from malware and impersonation attempts, please ensure you only download and install `async-cache-php` from our official sources:
- **Composer (Packagist):** `composer require fyennyi/async-cache-php`
- **GitHub Releases:** Official repository `https://github.com/Fyennyi/async-cache-php`

> [!WARNING]
> We **never** distribute this library as standalone `.exe` or `.zip` installers intended for manual execution. If you find a repository offering a "quick install" archive, it is likely malware. Please report it immediately.

## Supported Versions
Security updates are provided for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability
We take the security of this project very seriously. If you discover a security vulnerability, please **DO NOT** open a public issue. 

Instead, please report it privately:
1. Send an email directly to **chernegasergiy3@gmail.com**.
2. Or use the [GitHub Security Advisories](https://github.com/Fyennyi/async-cache-php/security/advisories) feature to report it privately.

Please provide a clear description of the issue and steps to reproduce it. You should receive a response within 48 hours. We will work with you to verify the vulnerability and release a patch as quickly as possible.
